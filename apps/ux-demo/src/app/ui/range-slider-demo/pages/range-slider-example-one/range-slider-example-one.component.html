Helper Info

min :
Minimum value of the slider.
max :
Maximum value of the slider.
selected :
Default selected value of the slider.
step:
Step difference of the slider.
Ex: 1) step="0.5" -- For 0.5 difference (Decimal Values) Ex: 2) step="2" -- for 2 step difference
disabled :
Set disability of the slider. ( Ex: disabled=true)
indicatorColor :
Set color of the indicator. ( Ex: indicatorColor="#2F80ED")
hideValues :
Set visibility of the numbers. ( Ex: hideValues=true)
hideDecimalValues :
Set visibility of decimal numbers (If step="0.5"). ( Ex: hideDecimalValues=true)
hideInputBox :
Set visibility of the input box. ( Ex: hideInputBox=true)
hideDescription :
Set visibility of the track's description. ( Ex: hideDescription=true)
tracks :
Array of the track with info like color, end step and description of the track.
Note For Accessibility : the disc(circle) will not slide left to right from keyboard keys, but we can switch focus on input field placed next to RATING text and on entering manual value there, the Disc will automatically slide to that value.
