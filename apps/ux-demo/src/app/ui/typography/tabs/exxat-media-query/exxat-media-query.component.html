Exxat MediaQuerry Examples
large screen : 22 px
extra-small screen : 12 px

To write the screen specific css you have to include the breakpoints.

Ex. @include media-breakpoint('xs')

Here the font size is 22 px for large screens. and 12 px for extra-small screens

You have to change the screen sizes in the brackets.

Note : Please refer the css on right hand side.

If you want same style for 1 or 2 screens then you can use the following example.

Ex. @include media-breakpoint('lt-md')

Here lt-md means "less than medium screens"

In this example the box color is pink for all the screens less than medium screen.and green for medium and the next screens.

Note : Please refer the css on right hand side.


If you want same style for 1 or 2 screens then you can use the following example.

Ex. @include media-breakpoint('gt-sm')

Here gt-sm means "greater than small screens"

In this example the box color is yellow for all the screens greater than small screen.and red for small and the bellow screens.

Note : Please refer the css on right hand side.

There are different breakpoint mixins are made in this theme

Ex. @include media-breakpoint('gt-sm')

In above example you cant replace "gt-sm" with the following values according to your screen.

xs , sm , md , lg , xl , lt-sm , lt-md , lt-lg , lt-xl , gt-xs , gt-sm , gt-md , gt-lg

Note : Please refer the css on right hand side.
